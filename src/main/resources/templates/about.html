All About Personal Project - Rate It Yourself (RIY)

This is a SoftUni Project for the Java Web - Spring Advanced module.
It's main purpose is to provide breakdown of food based on provided by user ingredients list so that they can visibly see and understand better what is in their pets food.  It should also provide easy way to browse foods, brand products, rate products, review, save favorite foods. etc
